# Скрипт для конвертации валют в другие валюты
## Описание
Этот проект представляет собой простой Python-скрипт для конвертации валют, использующий API v6.exchangerate-api.com.

##Установка
1. Клонируйте репозиторий:
```
git clone https://github.com/Qmndrn/api1.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
3. Получение и использование API-ключа
```
Зарегистрируйтесь и получите API-ключ на сайте exchangerate-api.com., чтобы получить API-ключ:
[[exchangerate-api.com]](https://www.exchangerate-api.com/)
```
далее в корне проекта создайте файл .env 
внутри файла:
 API_KEY = 'ваш_api_ключ'
```
##Использование
После установки зависимостей и создания файла .env запустите скрипт:
```
python main.py
-b, --base_currency — базовая валюта (по умолчанию RUB)

-t, --target_rate — целевая валюта (по умолчанию USD)

-m, --money — сумма для конвертации (по умолчанию 1000)
```
##Цель проекта:
Проект создан в образовательных целях и демонстрирует работу с API, переменными окружения, аргументами командной строки и JSON-ответами в Python.
